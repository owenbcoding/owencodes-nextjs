## The rebuild

I rebuilt this portfolio from the ground up using Next.js 16, the App Router, Tailwind v4, and MDX for content.

## Key decisions

- **Content as code**: Blogs live as MDX files, so writing a new post is just creating a file.
- **Server components first**: Pages render on the server whenever possible for speed.
- **Animated background**: The floating tech words give the site personality without stealing focus.

## Lessons learned

Start with the content model. Once the shape of a "blog" or "project" is clear, the UI writes itself.
